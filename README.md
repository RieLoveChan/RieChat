## English

RieChat is a clean, modern, and highly customizable Twitch chat overlay designed for streamers and content creators. It integrates seamlessly into broadcasting software like OBS Studio as a browser source.

### 🚀 Try It Instantly (No Installation Required!)
You can test and use RieChat immediately without downloading or installing anything! Simply use the hosted version on GitHub Pages:

*   **Clean URL (Recommended):**
    ```text
    https://rielovechan.github.io/RieChat/YourTwitchChannel
    ```
*   **Standard URL:**
    ```text
    https://rielovechan.github.io/RieChat/index.html?channel=YourTwitchChannel
    ```
*Replace `YourTwitchChannel` with your actual Twitch username.*

---

### ✨ Features
*   **Zero Installation:** Run it directly from GitHub Pages or open it as a local HTML file.
*   **Quick Connect:** Handy buttons to quickly connect and test with popular channels.
*   **Visual Customization:** Fully customize colors, fonts, margins, opacity, outlines, shadows, and rounded corners directly from the UI.
*   **Dynamic Animations:** Custom entry, stay, and exit animations for messages, usernames, and headers.
*   **Twitch Event Integration:** Supports displaying Cheers (bits), Channel Points redemptions, Subscription streaks, Raids, and Sub Gifts.
*   **OBS Friendly:** Automatically detects OBS Studio and makes the background completely transparent for a clean overlay.
*   **Import/Export Settings:** Copy your configuration as JSON or download/upload `settings.json` to keep your designs safe.

---

### ⚙️ How to Use

#### 1. Adding to OBS Studio
1.  Add a new **Browser Source** in your OBS scene.
2.  Set the URL to: `https://rielovechan.github.io/RieChat/index.html?channel=YourTwitchChannel` (or select **Local File** and point to your downloaded `index.html`).
3.  Set your desired width and height (e.g., `1920`x`1080` or a narrower aspect ratio depending on your overlay layout).
4.  RieChat will automatically remove its background and show only the styled chat messages.

#### 2. Saving and Loading Configurations
*   **Export:** Click the **Export Settings** button to download your `settings.json` file.
*   **Import:** Use the **Load from file** button or paste the JSON configuration in the text area at the bottom and click **Apply**.
*   **Reset:** Click **Restore Defaults** to revert all visual options.

## Español

RieChat es una superposición de chat de Twitch limpia, moderna y altamente personalizable diseñada para streamers y creadores de contenido. Se integra de manera fluida en programas de transmisión como OBS Studio como una fuente de navegador.

### 🚀 Pruébalo al Instante (¡Sin necesidad de instalar nada!)
¡Puedes probar y usar RieChat de inmediato sin descargar ni instalar nada en tu PC! Simplemente utiliza la versión alojada en GitHub Pages:

*   **URL Limpia (Recomendado):**
    ```text
    https://rielovechan.github.io/RieChat/TuCanalDeTwitch
    ```
*   **URL Estándar:**
    ```text
    https://rielovechan.github.io/RieChat/index.html?channel=TuCanalDeTwitch
    ```
*Reemplaza `TuCanalDeTwitch` por tu nombre de usuario de Twitch.*

---

### ✨ Características
*   **Instalación Cero:** Ejecútalo directamente desde GitHub Pages o ábrelo localmente como un archivo HTML.
*   **Conexión Rápida:** Botones dedicados para conectarte y hacer pruebas rápidamente con canales populares.
*   **Personalización Visual:** Personaliza colores, tipografías, márgenes, opacidad, bordes redondeados y sombras de texto directamente desde la interfaz gráfica.
*   **Animaciones Dinámicas:** Configura efectos de entrada, permanencia y salida para los mensajes, nombres de usuario y contenedores.
*   **Eventos de Twitch Integrados:** Muestra bits (cheers), canjes de Puntos de Canal, rachas de suscripción, raids y regalos de subs.
*   **Optimizado para OBS:** Detecta automáticamente OBS Studio y hace que el fondo sea transparente para integrarse de forma limpia sobre tus juegos.
*   **Importar/Exportar Configuración:** Copia tu configuración como JSON o descarga/sube tu archivo `settings.json` para no perder tus diseños.

---

### ⚙️ Cómo utilizarlo

#### 1. Agregar a OBS Studio
1.  Añade una nueva fuente de **Navegador** (Browser Source) en tu escena de OBS.
2.  Coloca la URL: `https://rielovechan.github.io/RieChat/TuCanalDeTwitch` (o marca la casilla de **Archivo local** y selecciona tu archivo `index.html` descargado).
3.  Configura el ancho y alto deseado (ej. `1920`x`1080` o un formato más angosto según tu diseño de transmisión).
4.  RieChat ocultará automáticamente el panel de control y el fondo para que los mensajes floten con transparencia.

#### 2. Guardar y Cargar Diseños
*   **Exportar:** Haz clic en el botón **Export Settings** para descargar tu archivo `settings.json` personalizado.
*   **Importar:** Usa el botón de **Cargar desde archivo** o pega el código JSON de tu configuración en la caja de texto inferior y haz clic en **Apply**.
*   **Restablecer:** Haz clic en **Restore Defaults** si quieres volver al estilo original.
